# Projeto de jogo

Sobre o jogo:


Um projeto sobre um jogo inspirado em guitar hero, um jogo que toca uma música e caem notas para o jogador clicar no momento certo e marcar sua pontuação


Funcionamento:

Sistema para fazer as notas cairem infinitamente enquanto a música estiver tocando, o jogador aperta a tecla "W" no momento certo para marcar um ponto, quando a musica acaba
é mostrado uma mensagem de fim de jogo e logo abaixo a pontuação do jogador


Oque falta para completar:

*Fazer as notas cairem no ritmo da musica

*Adicionar mais teclas para o jogador não ficar apenas com uma tecla
